SublimeText插件汇总

发表于 2017-08-12 | 分类于 前端

sublime也算是挺经典的编辑器了，以前编辑文字用一下，现在日常配合其他IDE食用。回想一下，竟然有很长一段时间我都不知道有插件这种东西存在。

ColorSublime

对于默认主题忍无可忍的旁友可以试一试，主题丰富任意挑选。

Emmet

被同学强势安利，前端必备，相对于刚开始写页面时是完全不同的体验。

Insert Nums

生成一列连续的数字，某些特定情况下配合st的强大操作使用。

AutoFileName

经历过从Dreamweaver到其他编辑器的同学一定对文件路径很是头疼，这个插件在写文件路径是效果很不错。

ConvertToUTF8

乱码什么的，不存在的
